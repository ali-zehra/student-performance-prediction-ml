# Student Performance Prediction using Machine Learning

## Overview
This project predicts student academic performance based on demographic and academic features using machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset
Student Performance Dataset from UCI Machine Learning Repository.

## Approach
- Data preprocessing
- Feature selection
- Model training using regression models
- Model evaluation

## Model Used
- Linear Regression

## Conclusion
The model provides insights into factors affecting student performance and demonstrates the application of machine learning in educational data analysis.
